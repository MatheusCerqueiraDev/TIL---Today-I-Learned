Comandos linux

abrir o terminal - ctrl + alt + t
clear - limpa o terminal
pwd - saber a pasta atual
cd - acessar pastas ou diretório
tab - você visualiza o que pode ser acessado
seta para direita - completa o autofill quer aparace
seta para cima - volta os comandos
ls - lista arquivos e pastas
mkdir - cria uma pasta 
rmdir - deleta uma pasta
touch - cria arquivo, é possível criar mais de um arquivo por vez
rm -deleta arquivo
cat - lê o que existe dentro de um arquivo
cp - clona um arquivo e renomeia
mv - renomeia um arquivo de um diretório para o outro
head - lê o que existe dentro de um arquivo
tail - mostra as ultimas dez linhas do arquivo
tac - lê o que existe dentro de um arquivo ao contrario
more - mostra todo o conteudo do arquivo
